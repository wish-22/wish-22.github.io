Other Notes
Heart Routing
Heart obtains are based on typical runs, sourced ethically from Speedrun Dot Com.

Core requires four hearts to unlock.
The quickest hearts are from the first four chapters.
However, some runs get hearts from B-sides
Temple B-Side
Due to its short length, Temple B-Side is actually a viable strategy in Any%-based runs. Its only caveat is that it's more difficult than the A-Side.
Reflection B-Side
6B is also a viable strategy in Any%-based runs. Also like 5B, it's difficult, to the point where it isn't highly recommended for newer runners to use. However, it's required when playing at high levels.

In ARB, 6B is also useful for getting a Crystal Heart during a level that doesn't have any berries.
Farewell with < 15 Hearts
In Farewell before Event Horizon, a 15-heart Gate drops from the sky.

Getting 15 hearts isn't necessary! Rather than spend time completing every A-Side and B-side, getting enough speed allows you to skip the gate! You can see an example here.
Cheat Mode
Some runs require the activation of Cheat Mode to play. This is done in Prologue.
The code is as follows: Left, Right, Journal, Grab, Up, Up, Down, Left, Grab, Confirm.
All directional inputs are menu inputs.

You can also set up keybinds that you hit in order, so that you don't have to remember the code - for example, I have these on keys 1-6, and I just swipe my finger across all of them.
In order, they are:
  1. Left
  2. Right + Journal
  3. Grab
  4. Up
  5. Up + Left + Down + Grab
  6. Confirm
Cheat Mode unlocks every level, checkpoint, and heart gate.
